= What is This

This is a plugin of iOS Simulator that send MemoryWarning repeatedly (every 5sec, 15sec, 60sec).

MemoryWarnings in the users iOS device happen unpredictably, so it helps testing such situation.

= Requirements

* SIMBL(0.9.9 or later): http://www.culater.net/software/SIMBL/SIMBL.php

* tested only
** MacOS X Mountain Lion 10.8.2
** XCode 4.5

= Install

Run MemoryWarningSender.pkg, install into "~/Library/Application\ Support/SIMBL/Plugins/MemoryWarningSender.bundle" 

= How To Use

iOS Simulator's Menu -> Hardware -> "Send MemoryWarning Repeatedly" -> <select mode>
